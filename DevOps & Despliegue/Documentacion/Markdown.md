# Markdown

Markdown es un lenguaje de marcado ligero usado para escribir documentos fácilmente formateados. Es ideal para README, documentación técnica, blogs, etc.

---

## 1. Encabezados

```markdown
# Título 1

## Título 2

### Título 3

#### Título 4
```

## 2. Texto

```markdown
**Negrita** => **Negrita**
_Itálica_ => _Itálica_
~~Tachado~~ => ~~Tachado~~
`Código` => `Código`
```

## 3. Listas

### 3.1. Lista con viñetas

```markdown
- Item 1
- Item 2
  - Subitem
```

### 3.2. Lista numerada

```markdown
1. Paso uno
2. Paso dos
   1. Subpaso
```
